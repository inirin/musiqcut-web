파이프라인 가이드



      

        
작품 만들기

        

          테마/스토리만 입력하면 AI가 분위기, 아트 스타일, 캐릭터, 음악까지 모두 자동으로 결정합니다.
        

        

          
숏폼15~30초 · 가사 2~3줄 · 바로 보컬 시작 · 다채로운 구성

        

      


      

        

          
✍️ STEP 1 — 스토리/컨셉 생성

          
Gemini 2.5 Flash + Google Search grounding · Google Cloud API

          

            
입력 테마 + 분위기 + 트렌드 힌트 (자동 생성 시)

            
출력 제목, 가사, 작곡 지시, 보컬 스타일, 아트 스타일, 캐릭터 프로필

          

          

            테마에서 분위기/아트 스타일을 자동 판단하고, 캐릭터 외형 프로필(최대 3명)을 상세 정의합니다.
            트렌드 힌트가 있으면 Google 검색으로 실존 인물/작품을 파악하여 캐릭터 외형·보컬·아트 스타일에 반영합니다 (고유명사는 제목/가사에 사용하지 않음).
          

        

        

          
🎵 STEP 2 — 음악 생성

          
Suno AI v3.5 · 외부 API

          

            
입력 STEP 1의 스토리/컨셉 + 작곡 지시

            
출력 MP3 음원 + 단어별 정밀 타이밍 + 5초 단위 세그먼트 → 장면 수 자동 결정

          

          

            Suno v3.5가 작곡 → 인트로 자동 트림(보컬 시작점 감지) → 15~30초.
            Demucs 보컬 분리 → Whisper large-v3 전사 + whisperx forced alignment(wav2vec2)로 단어별 정밀 타이밍 추출 → Gemini Flash가 오타 보정.
            보컬 감지 실패(50% 미만) 시 Step 1부터 새 스토리/가사/음악으로 자동 재시도 (최대 2회).
          

        

        

          
🖼️ STEP 3 — 이미지 생성

          
Gemini 2.5 Flash (장면 구성) + Imagen 4 (T2I) · Google Cloud API

          

            
입력 STEP 1의 스토리 + STEP 2에서 결정된 장면 수

            
출력 장면별 설명/프롬프트 + PNG 이미지 (576×1024 세로 9:16)

          

          

            Whisper 가사 싱크 타이밍 + 캐릭터 프로필을 기반으로 장면을 구성합니다.
            STEP 1에서 결정된 아트 스타일이 모든 장면에 일관 적용됩니다.
          

        

        

          
🎞️ STEP 4 — 영상 클립 생성

          
Wan 2.2 S2V (립싱크) + Wan 2.2 I2V (모션) · ComfyUI · 로컬 GPU

          

            
입력 STEP 3의 이미지 + 장면 설명 + 보컬 오디오

            
출력 장면별 MP4 영상 클립 (576×1024 9:16, ~5초)

          

          

            Whisper 보컬 타이밍 + 보컬 주체 여부(is_vocalist) + 샷 타입으로 엔진 자동 분기.
            보컬 주체의 클로즈업/미디엄은 Wav2Vec2 기반 립싱크, 나머지는 모션 영상.
          

          

            
엔진 분기

            보컬 주체 + 클로즈업/미디엄 → Wan 2.2 S2V (Wav2Vec2 오디오 립싱크, ~17분/클립)

            와이드/비보컬/비주체 → Wan 2.2 I2V (환경 모션, Lightning LoRA 고속)
          

          

            
폴백 전략

            S2V 실패 → 정지 이미지 영상 · Wan I2V 실패 → 정지 이미지 영상
          

        

        

          
🎬 STEP 5 — 최종 영상 합성

          
FFmpeg · 로컬

          

            
입력 STEP 4의 영상 클립 + STEP 2의 음원

            
출력 최종 MP4 뮤직비디오

          

          

            클립 해상도 정규화 (736×1280) + 장면 duration 정밀 trim + FPS 통일 (24fps) → 오디오 합성 + 마지막 2초 fade-out → faststart 최적화.

  • 완성된 작품에 👍/👎 + 피드백 가능
    • 
          
  • 각 스텝(스토리/음악)과 클립별로도 피드백 가능
    • 
        

      

      

        
⏰ 자동 생성 스케줄러

        
    
          
  • 설정한 간격(기본 2시간)마다 Google Trends 실시간 키워드 기반으로 테마를 생성하여 자동 작품 제작
    • 
          
  • Gemini Google Search grounding으로 트렌드 맥락 파악 → 구체적 상황 기반 테마 생성
    • 
          
  • 서버 재시작 시 중단된 작품 자동 resume, 보컬 감지 실패 시 Step 1부터 자동 재시도
    • 
          
  • 설정 페이지에서 활성화/간격 조정 가능
    • 
        

      


      

        
참고 사항

        
    
          
  • 한 번에 하나의 작업만 실행 가능합니다 (GPU/API 보호)
    • 
          
  • 실패한 스텝부터 재시도할 수 있으며, 이전 스텝의 결과는 캐시됩니다
    • 
          
  • 이전 스텝부터 재시도하면 이후 스텝은 자동으로 새로 생성됩니다
    • 
          
  • STEP 4 진행 중 리소스 모니터로 GPU 활용 상태를 확인할 수 있습니다
    • 
          
  • 설정 페이지에서 API 키 연결 상태를 확인하세요
