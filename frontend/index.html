파이프라인 가이드



      

        
작품 만들기

        

          테마/스토리만 입력하면 AI가 분위기, 아트 스타일, 캐릭터, 음악까지 모두 자동으로 결정합니다.
        

        

          
숏폼15~30초 · 가사 2~3줄 · 바로 보컬 시작 · 다채로운 구성

        

      


      

        

          
✍️ STEP 1 — 스토리/컨셉 생성

          

            
입력 테마 + 분위기 + 트렌드 힌트 (자동 생성 시)

            
출력 제목, 가사, 작곡 지시, 보컬 스타일, 아트 스타일, 캐릭터 프로필, 해시태그

          

          

            테마에서 장르/분위기/아트 스타일을 자동 판단하고, 캐릭터 외형 프로필(최대 3명)을 상세 정의합니다.
            트렌드 힌트가 있으면 검색으로 대상(인물/사물/장소 등)의 실제 외형을 파악하여 캐릭터/소품/배경에 반영합니다.
            업로드용 해시태그 5개도 함께 생성됩니다.
          

        

        

          
🎵 STEP 2 — 음악 생성

          

            
입력 STEP 1의 가사 + 작곡 지시

            
출력 MP3 음원 + 단어별 정밀 타이밍 + 장면 수 자동 결정

          

          

            AI 작곡 → 인트로 자동 트림(보컬 시작점 감지) → 15~30초.
            보컬 분리 → 음성 인식으로 단어별 정밀 타이밍 추출 → AI가 오타 보정.
            보컬 감지 실패(50% 미만) 시 Step 1부터 새 스토리/가사/음악으로 자동 재시도 (최대 2회).
          

        

        

          
🖼️ STEP 3 — 이미지 생성

          

            
입력 STEP 1의 스토리 + STEP 2에서 결정된 장면 수

            
출력 장면별 설명/프롬프트 + PNG 이미지 (576×1024 세로 9:16)

          

          

            가사 타이밍 + 캐릭터 프로필을 기반으로 장면을 구성합니다.
            STEP 1에서 결정된 아트 스타일이 모든 장면에 일관 적용됩니다.
          

        

        

          
🎞️ STEP 4 — 영상 클립 생성

          

            
입력 STEP 3의 이미지 + 장면 설명 + 보컬 오디오

            
출력 장면별 MP4 영상 클립 (576×1024 9:16, ~5초)

          

          

            보컬 타이밍 + 보컬 주체 여부(is_vocalist)로 엔진 자동 분기.
          

          

            
엔진 분기

            보컬 + 보컬 주체 → 오디오 립싱크 (~17분/클립)

            비보컬/비주체 → 모션 영상 (고속)
          

          

            
폴백 전략

            립싱크 실패 → 정지 이미지 영상 · 모션 실패 → 정지 이미지 영상
          

        

        

          
🎬 STEP 5 — 최종 영상 합성

          

            
입력 STEP 4의 영상 클립 + STEP 2의 음원

            
출력 최종 MP4 뮤직비디오

          

          

            클립 해상도 정규화 (736×1280) + 장면 trim + 24fps 통일 → 오디오 합성 + fade-out.
            제목/테마 자막 (어그로체) + 노래방 스타일 가사 자막 (카페24 단정해).
          

        

      


      

        
💬 피드백 시스템

        
    
          
  • 완성된 작품에 👍/👎 + 피드백 가능
    • 
          
  • 각 스텝(스토리/음악)과 클립별로도 피드백 가능
    • 
        

      

      

        
⏰ 자동 생성 스케줄러

        
    
          
  • 설정한 간격(기본 2시간)마다 실시간 트렌드/뉴스 기반으로 테마를 생성하여 자동 작품 제작
    • 
          
  • 검색으로 트렌드 맥락 파악 → 이야기 잠재력 기준 소재 선택 (니치/전문 소재도 가능)
    • 
          
  • 최근 30개 작품과 중복 소재 자동 방지 (같은 뉴스 영감 차단)
    • 
          
  • 서버 재시작 시 중단된 작품 자동 resume, 스케줄러 비정상 종료 시 자동 복구
    • 
          
  • 설정 페이지에서 활성화/간격 조정 가능
    • 
        

      

      

        
📤 업로드 시스템

        
    
          
  • YouTube, Instagram, TikTok 계정 연동 후 수동/자동 업로드
    • 
          
  • 플랫폼별 제목 포맷 자동 적용 (YouTube: 하이픈 구분, Instagram/TikTok: 줄바꿈)
    • 
          
  • 해시태그가 설명(description)에 자동 포함
    • 
          
  • 중복 업로드 방지 (플랫폼별 1회)
    • 
        

      


      

        
참고 사항

        
    
          
  • 한 번에 하나의 작업만 실행 가능합니다 (GPU/API 보호)
    • 
          
  • 실패한 스텝부터 재시도할 수 있으며, 이전 스텝의 결과는 캐시됩니다
    • 
          
  • 이전 스텝부터 재시도하면 이후 스텝은 자동으로 새로 생성됩니다
    • 
          
  • 특정 장면만 재생성 가능 (이미지+클립 또는 클립만)
    • 
          
  • STEP 4 진행 중 리소스 모니터로 GPU 활용 상태를 확인할 수 있습니다
    • 
          
  • 도메인: musiqcut.com (Cloudflare Tunnel + Access 인증)
    •
