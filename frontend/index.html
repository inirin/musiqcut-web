사용 가이드


      

        
뮤직비디오 만들기

        

          테마만 입력하면 AI가 스토리, 음악, 이미지, 영상, 자막까지 모두 자동으로 만들어줍니다.

          완성까지 약 30~50분 소요되며, 진행 상황을 실시간으로 확인할 수 있습니다.
        

        

          
영상 길이15~30초 숏폼

          
해상도세로 9:16 (모바일 최적)

        

      


      

        

          
✍️ STEP 1 — 스토리 만들기

          

            입력한 테마를 바탕으로 AI가 제목, 가사, 캐릭터 디자인, 그림체, 음악 스타일을 결정합니다.
            업로드용 해시태그도 함께 생성됩니다.
          

        

        

          
🎵 STEP 2 — 음악 만들기

          

            AI가 가사에 맞는 음악을 작곡하고, 가사의 단어별 타이밍을 분석합니다.
            이 타이밍이 자막과 장면 전환의 기준이 됩니다.
          

        

        

          
🖼️ STEP 3 — 장면 이미지 생성

          

            가사 흐름에 맞춰 장면을 구성하고, 각 장면의 이미지를 생성합니다.
            STEP 1에서 정한 그림체와 캐릭터가 모든 장면에 일관되게 적용됩니다.
          

        

        

          
🎞️ STEP 4 — 영상 클립 생성

          

            각 이미지를 ~5초짜리 움직이는 영상으로 변환합니다.
            노래하는 캐릭터 장면은 입 모양이 음악에 맞춰 움직이는 립싱크가 적용됩니다.
          

          

            
소요 시간

            립싱크 장면: 약 17분/클립 · 일반 장면: 약 2분/클립
          

        

        

          
🎬 STEP 5 — 최종 영상 완성

          

            모든 클립을 이어붙이고 음악을 합성하여 최종 뮤직비디오를 완성합니다.
            제목/테마 자막과 노래방 스타일 가사 자막이 자동으로 들어갑니다.
          

        

      


      

        
🔄 재생성

        
    
          
  • 마음에 안 드는 장면을 클릭하여 해당 장면만 다시 만들 수 있습니다
    • 
          
  • 이미지만 재생성하거나, 이미지+영상 클립 모두 재생성할 수 있습니다
    • 
          
  • 영상 생성 중에도 다른 장면의 재생성을 요청할 수 있습니다 (대기 후 자동 처리)
    • 
          
  • 실패한 단계부터 재시도할 수 있으며, 이전 단계 결과는 유지됩니다
    • 
        

      


      

        
⏰ 자동 생성

        
    
          
  • 설정에서 켜면, 정해진 간격마다 실시간 트렌드/뉴스를 기반으로 자동 작품 제작
    • 
          
  • 같은 소재가 중복 선택되지 않도록 최근 30개 작품을 체크합니다
    • 
          
  • 생성 간격은 설정 페이지에서 조정할 수 있습니다
    • 
        

      


      

        
📤 채널 업로드

        
    
          
  • 설정에서 YouTube, Instagram, TikTok 계정을 연결할 수 있습니다
    • 
          
  • 완성된 작품을 원하는 플랫폼에 수동 업로드하거나, 자동 업로드를 켤 수 있습니다
    • 
          
  • 제목과 해시태그가 플랫폼에 맞게 자동 포맷됩니다
    • 
          
  • 같은 작품은 플랫폼당 한 번만 업로드됩니다
    • 
        

      


      

        
💬 피드백

        
    
          
  • 완성된 작품에 👍/👎 + 의견을 남길 수 있습니다
    • 
          
  • 각 단계(스토리/음악)와 개별 클립에도 피드백 가능합니다
    • 
        

      


      

        
알아두면 좋은 것

        
    
          
  • 한 번에 하나의 작품만 생성할 수 있습니다
    • 
          
  • STEP 4(영상 클립) 진행 중 GPU 사용량을 실시간 모니터링할 수 있습니다
    • 
          
  • 작품 정보에서 해시태그를 클릭하면 텍스트가 선택되어 복사하기 편합니다
    • 
          
  • 설정 페이지에서 API 키 연결 상태를 확인하세요
    •
