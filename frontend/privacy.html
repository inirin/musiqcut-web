뮤직컷 (MusiqCut) 개인정보처리방침

최종 수정일: 2026년 3월 28일

뮤직컷(musiqcut.com)은 AI 기반 숏폼 뮤직비디오 자동 생성 서비스입니다. 본 방침은 서비스 이용 과정에서 수집되는 정보와 그 처리에 대해 안내합니다.

1. 수집하는 정보

  • 인증 정보: Cloudflare Access를 통한 Google 로그인 시 이메일 주소가 인증 목적으로 사용됩니다. 이 정보는 Cloudflare에서 관리하며 본 서비스 서버에 저장되지 않습니다.
  • 소셜 미디어 연동 정보: YouTube, Instagram, TikTok 계정 연동 시 해당 플랫폼의 OAuth 토큰(접근 토큰, 갱신 토큰)과 채널/계정 이름을 서버 로컬 데이터베이스에 저장합니다.
  • API 키: 사용자가 입력한 외부 서비스 API 키(Google Gemini, Suno AI)를 서버 설정 파일에 저장합니다.

2. 정보의 이용 목적

  • 소셜 미디어 토큰: 사용자가 요청한 영상 업로드 기능 수행
  • API 키: AI 뮤직비디오 생성 파이프라인 실행 (스크립트 생성, 음악 생성, 이미지/영상 생성)

3. 외부 서비스 연동

본 서비스는 다음 외부 서비스의 API를 사용합니다:

  • Google Gemini API: 스크립트/장면 생성, 이미지 생성 (Imagen 4)
  • Suno AI: 음악 생성
  • YouTube Data API: 영상 업로드
  • Instagram Graph API: 릴스 업로드
  • TikTok API: 영상 업로드
  • Cloudflare Access: 접근 인증

각 서비스로 전송되는 데이터는 해당 서비스의 개인정보처리방침을 따릅니다.

4. 정보의 보관 및 삭제

  • 소셜 미디어 연동 해제 시 저장된 토큰과 계정 정보는 즉시 삭제됩니다.
  • 생성된 작품(영상, 이미지, 음악)은 서버 로컬 스토리지에 보관되며, 작품 삭제 시 모든 관련 파일이 함께 삭제됩니다.
  • 업로드 이력(영상 ID, URL)은 서비스 이용 기록으로 보관됩니다.

5. 제3자 제공

수집된 정보는 제3자에게 제공되지 않습니다. 영상 업로드 시 해당 플랫폼 API를 통해 직접 전송됩니다.

6. 데이터 삭제 요청

설정 페이지에서 플랫폼 연결 해제 및 작품 삭제가 가능합니다. 전체 데이터 삭제를 원하시면 서비스 관리자에게 연락해주세요.
