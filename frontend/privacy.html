뮤직컷 (MusiqCut) 개인정보처리방침

최종 수정일: 2026년 3월 27일

1. 수집하는 정보

소셜 미디어 계정 연동 시 해당 플랫폼의 OAuth 토큰(접근 토큰, 갱신 토큰)과 채널/계정 이름을 저장합니다. 그 외 개인정보는 수집하지 않습니다.

2. 정보의 이용 목적

수집된 정보는 오직 사용자가 요청한 영상 업로드 기능을 수행하기 위해서만 사용됩니다.

3. 정보의 보관 및 삭제

연동 해제 시 저장된 토큰과 계정 정보는 즉시 삭제됩니다. 업로드 이력(영상 ID, URL)은 서비스 이용 기록으로 보관됩니다.

4. 제3자 제공

수집된 정보는 제3자에게 제공되지 않습니다. 영상 업로드 시 해당 플랫폼(YouTube, Instagram, TikTok)의 API를 통해 직접 전송됩니다.

5. 데이터 삭제 요청

설정 페이지에서 플랫폼 연결 해제 시 모든 인증 정보가 삭제됩니다.

6. 문의

개인정보 관련 문의는 서비스 관리자에게 연락해주세요.
