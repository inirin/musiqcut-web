뮤직컷 (MusiqCut) 개인정보처리방침

시행일: 2026년 3월 30일 | 최종 수정일: 2026년 3월 30일

뮤직컷(이하 "서비스")은 musiqcut.com에서 운영되는 AI 기반 숏폼 뮤직비디오 자동 생성 서비스입니다. 본 개인정보처리방침은 서비스 이용 과정에서 수집되는 개인정보의 항목, 수집 목적, 보관 기간, 제3자 제공 여부 및 이용자의 권리에 대해 안내합니다.

1. 수집하는 개인정보의 항목 및 수집 방법

1.1 자동 수집 항목

  • 인증 정보: Cloudflare Access를 통한 Google 로그인 시 이메일 주소가 접근 인증 목적으로 사용됩니다. 이 정보는 Cloudflare에서 관리되며, 서비스 서버에 별도 저장되지 않습니다.
  • 접속 로그: 서비스 이용 시 IP 주소, 접속 시간, 브라우저 정보가 서버 로그에 기록될 수 있습니다. 이는 서비스 안정성 및 보안 목적으로만 사용됩니다.

1.2 이용자 제공 항목

  • 소셜 미디어 연동 정보: YouTube, Instagram, TikTok 계정 연동 시 해당 플랫폼의 OAuth 토큰(접근 토큰, 갱신 토큰), 채널/계정 식별자 및 표시 이름을 서버 로컬 데이터베이스에 저장합니다.
  • API 키: 사용자가 입력한 외부 AI 서비스 API 키를 서버 환경 설정 파일에 저장합니다.
  • 생성 콘텐츠: 서비스를 통해 생성된 영상, 이미지, 음악, 가사, 스크립트 등의 콘텐츠 파일이 서버 로컬 스토리지에 보관됩니다.

2. 개인정보의 이용 목적

  • 서비스 접근 인증 및 권한 관리
  • 소셜 미디어 플랫폼으로의 영상 업로드 기능 수행
  • AI 뮤직비디오 생성 파이프라인 실행
  • 서비스 오류 진단 및 성능 개선
  • 서비스 이용 통계 분석 (비식별 처리)

3. 개인정보의 보관 기간 및 파기

  • 소셜 미디어 토큰: 연동 해제 시 즉시 삭제됩니다. 플랫폼별 토큰 만료 정책에 따라 자동 갱신 또는 만료될 수 있습니다.
  • 생성 콘텐츠: 작품 삭제 시 관련 모든 파일(영상, 이미지, 음악, 메타데이터)이 즉시 삭제됩니다.
  • 업로드 이력: 영상 ID, 플랫폼 URL 등의 업로드 기록은 서비스 이용 기록으로 보관되며, 작품 삭제 시 함께 삭제됩니다.
  • 서버 로그: 접속 로그는 최대 30일간 보관 후 자동 삭제됩니다.

4. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 외부에 판매, 대여 또는 제공하지 않습니다. 각 플랫폼의 데이터는 아래 명시된 목적으로만 사용되며, 목적 외 용도로 저장하거나 활용하지 않습니다.

  • YouTube (Google): 사용자가 요청한 영상 업로드를 위해 YouTube API를 통해 영상 파일, 제목, 설명, 태그를 전송합니다. YouTube 사용자 데이터는 영상 업로드 목적으로만 사용되며, 서비스 내에 별도 저장하지 않습니다. 연동 해제 시 저장된 OAuth 토큰은 즉시 삭제됩니다.
  • Instagram (Meta): 사용자가 요청한 영상 업로드를 위해 Instagram API를 통해 영상 파일, 캡션을 전송합니다. Instagram 사용자 데이터는 영상 업로드 목적으로만 사용되며, 서비스 내에 별도 저장하지 않습니다. 연동 해제 시 저장된 OAuth 토큰은 즉시 삭제됩니다.
  • TikTok: 사용자가 요청한 영상 업로드를 위해 TikTok Content Posting API를 통해 영상 파일, 제목을 전송합니다. TikTok 사용자 데이터는 영상 업로드 목적으로만 사용되며, 서비스 내에 별도 저장하지 않습니다. 연동 해제 시 저장된 OAuth 토큰은 즉시 삭제됩니다.
  • AI 서비스: 콘텐츠 생성 시 외부 AI API로 텍스트 프롬프트가 전송됩니다. 사용자의 개인정보는 AI 서비스로 전송되지 않습니다.

각 플랫폼으로 전송된 데이터는 해당 서비스의 개인정보처리방침 및 이용약관을 따릅니다.

5. 개인정보의 안전성 확보 조치

  • HTTPS 암호화 통신 (Cloudflare Tunnel)
  • Cloudflare Access를 통한 접근 제어 (Google 인증 필수)
  • OAuth 토큰은 서버 로컬 데이터베이스에 암호화 없이 저장되나, 서버 접근 자체가 인증으로 보호됩니다.
  • API 키는 서버 환경 파일에 저장되며, 외부에서 접근할 수 없습니다.

6. 이용자의 권리

  • 연동 해제: 설정 페이지에서 언제든지 소셜 미디어 계정 연동을 해제할 수 있습니다.
  • 작품 삭제: 대시보드에서 생성된 작품과 관련 데이터를 삭제할 수 있습니다.
  • 전체 데이터 삭제: 서비스에 저장된 모든 데이터의 삭제를 요청할 수 있습니다.
  • 정보 열람: 본인의 개인정보 처리 현황에 대해 열람을 요청할 수 있습니다.

7. 쿠키 및 추적 기술

본 서비스는 자체적으로 쿠키를 사용하지 않습니다. 다만, Cloudflare Access 인증 과정에서 Cloudflare가 세션 쿠키를 사용할 수 있으며, 이는 Cloudflare의 개인정보처리방침을 따릅니다.

8. 아동의 개인정보 보호

본 서비스는 만 14세 미만의 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

9. 개인정보처리방침의 변경

본 방침이 변경되는 경우 시행일 최소 7일 전에 서비스 내 공지를 통해 안내합니다. 중대한 변경의 경우 30일 전에 안내합니다.

MusiqCut Privacy Policy

Effective: March 30, 2026 | Last updated: March 30, 2026

MusiqCut ("Service") is an AI-powered short-form music video generation service operated at musiqcut.com. This Privacy Policy describes the personal information we collect, why we collect it, how we store and protect it, and your rights regarding your data.

1. Information We Collect

1.1 Automatically Collected

  • Authentication Data: Email addresses are used for access authentication via Cloudflare Access (Google login). This information is managed by Cloudflare and is not stored on our servers.
  • Access Logs: IP addresses, access times, and browser information may be recorded in server logs for security and service stability purposes.

1.2 User-Provided

  • Social Media Integration: When connecting YouTube, Instagram, or TikTok accounts, OAuth tokens (access/refresh tokens), channel/account identifiers, and display names are stored in our local database.
  • API Keys: External AI service API keys entered by users are stored in server configuration files.
  • Generated Content: Videos, images, music, lyrics, and scripts created through the service are stored on local server storage.

2. How We Use Your Information

  • Service access authentication and authorization
  • Uploading videos to social media platforms on your behalf
  • Running the AI music video generation pipeline
  • Diagnosing errors and improving service performance
  • Analyzing service usage statistics (anonymized)

3. Data Retention and Deletion

  • Social Media Tokens: Immediately deleted when you disconnect a platform. Tokens may auto-refresh or expire per each platform's policy.
  • Generated Content: All related files (video, images, music, metadata) are immediately deleted when a project is deleted.
  • Upload History: Upload records (video IDs, platform URLs) are retained as service usage records and deleted along with the project.
  • Server Logs: Access logs are retained for up to 30 days and then automatically deleted.

4. Third-Party Data Sharing

We do not sell, rent, or share your personal information with third parties. Data from each platform is used solely for the purposes described below and is not stored or used for any other purpose.

  • YouTube (Google): Video files, titles, descriptions, and tags are sent via the YouTube API for user-requested uploads. YouTube user data is used solely for video uploading and is not stored separately within the service. OAuth tokens are immediately deleted upon disconnection.
  • Instagram (Meta): Video files and captions are sent via the Instagram API for user-requested uploads. Instagram user data is used solely for video uploading and is not stored separately within the service. OAuth tokens are immediately deleted upon disconnection.
  • TikTok: Video files and titles are sent via the TikTok Content Posting API for user-requested uploads. TikTok user data is used solely for video uploading and is not stored separately within the service. OAuth tokens are immediately deleted upon disconnection.
  • AI Services: Text prompts are sent to external AI APIs during content generation. No personal user information is transmitted to AI services.

Data sent to these services is governed by their respective privacy policies and terms of service.

5. Data Security

  • All communications are encrypted via HTTPS (Cloudflare Tunnel)
  • Access control via Cloudflare Access (Google authentication required)
  • OAuth tokens are stored in a local database; server access itself is protected by authentication
  • API keys are stored in server environment files and are not externally accessible

6. Your Rights

  • Disconnect Accounts: You can disconnect social media accounts at any time from the settings page.
  • Delete Projects: You can delete generated projects and all associated data from the dashboard.
  • Complete Data Deletion: You may request deletion of all data stored in the service.
  • Access Information: You may request to view your personal data processing status.

7. Cookies and Tracking

This service does not use cookies on its own. However, Cloudflare Access may use session cookies during the authentication process, governed by Cloudflare's privacy policy.

8. Children's Privacy

This service is not directed at children under 14 years of age, and we do not knowingly collect personal information from children.

9. Changes to This Policy

If this policy is updated, we will provide notice through the service at least 7 days before the effective date. For significant changes, we will provide 30 days' notice.

10. Contact Us

Privacy inquiries and data deletion requests: musiqcut@gmail.com
